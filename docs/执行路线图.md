  - 打开本地 demo 页面。

  - 点击 `A+` 后字号变大。

当前进度可理解为 v0.1 MVP 的约 15%。

## 1. 总体目标

v0.1 MVP 的目标是：

```text
用户打开一个 HTML 页面
  -> 启用 ClickDeck 插件
  -> 点选页面元素
  -> 用语义化控件微调视觉样式
  -> 可撤销/重做
  -> 可复制诊断信息
  -> 导出 HTML 快照
  -> 导出 PDF
```

非目标：

- 不接入 AI。
- 不生成内容。
- 不回写源码。
- 不做 Figma 式自由拖拽。
- 不做可编辑 PPT 导出。

## 5. 任务 2：建立编辑状态层

### 目标

当前代码直接改 DOM，没有稳定状态模型。后续撤销、导出、诊断都需要状态层。

### 需要建立的数据结构

建议先从简单结构开始：

```ts
type SelectedElementState = {
  element: HTMLElement;
  descriptor: string;
};

type StylePatch = {
  id: string;
  targetDescriptor: string;
  property: "fontSize" | "textAlign" | "color" | "margin" | "padding";
  before: string;
  after: string;
  createdAt: number;
};

type EditorState = {
  active: boolean;
  selected: SelectedElementState | null;
  patches: StylePatch[];
};
```

### 注意点

- 不要一开始追求完美 selector。
- `HTMLElement` 可以保存在运行时状态里，但导出/诊断要用可序列化 descriptor。
- descriptor 先用 `tag#id.class1.class2` 即可，后续再增强。

### 验收标准

- 选中元素会进入 state。
- 每次样式修改都会记录 patch。
- 当前行为不退化。
- E2E 仍通过。

## 6. 任务 3：撤销/重做

### 目标

任何会修改页面样式的动作都必须可撤销，这是用户安全感的核心。

### 功能范围

MVP 先支持：

- Undo：撤销上一次样式修改。
- Redo：恢复刚撤销的样式修改。
- 快捷键：
  - Windows/Linux: `Ctrl+Z`, `Ctrl+Shift+Z`
  - macOS 后续再补 `Meta+Z`

### 实现方向

每个 `StylePatch` 记录：

- 目标元素引用。
- 目标元素 descriptor。
- property。
- before。
- after。

Undo 时把 `before` 写回。

Redo 时把 `after` 写回。

### UI

面板上增加：

- Undo 按钮。
- Redo 按钮。

按钮在不可用时 disabled。

### 验收标准

- 点击 `A+` 后字号变大。
- Undo 后字号恢复。
- Redo 后字号再次变大。
- E2E 增加对应验证。

## 7. 任务 4：诊断日志缓冲

### 目标

现在 logger 只写 console。需要建立内存日志缓冲，后续用于“复制诊断信息”。

### 日志结构

```ts
type LogEntry = {
  id: string;
  level: "debug" | "info" | "warn" | "error";
  context: "extension" | "selection" | "style" | "export" | "diagnostics";
  message: string;
  details?: unknown;
  createdAt: number;
};
```

### 功能

- 保留最近 100 条日志。
- logger 仍然写 console。
- logger 同时写入内存 buffer。
- 提供 `getRecentLogs()`。
- 提供 `clearLogs()`。

### 后续用途

- UI 上增加“Copy diagnostics”。
- E2E 失败时可以快速知道哪一步失败。
- 用户提 issue 时可以复制诊断。

### 验收标准

- logger 单元测试覆盖 buffer 写入。
- 不影响现有日志输出。

// --- Drag logic start ---
const header = element.querySelector<HTMLElement>(".clickdeck-panel__header");
if (header) {
  let isDragging = false;
  let offsetX = 0;
  let offsetY = 0;

  header.addEventListener("mousedown", (e: MouseEvent) => {
    // Ignore if clicking a button inside the header
    if ((e.target as HTMLElement).closest("button")) {
      return;
    }
    isDragging = true;
    offsetX = e.clientX - element.getBoundingClientRect().left;
    offsetY = e.clientY - element.getBoundingClientRect().top;
    e.preventDefault();
  });

  window.addEventListener("mousemove", (e: MouseEvent) => {
    if (!isDragging) {
      return;
    }
    const x = e.clientX - offsetX;
    const y = e.clientY - offsetY;
    element.style.left = `${x}px`;
    element.style.top = `${y}px`;
    element.style.right = "auto";
  });

  window.addEventListener("mouseup", () => {
    isDragging = false;
  });
}
// --- Drag logic end ---

case "pick-bg-color": {

  let bg = "";
  let current: HTMLElement | null = element.parentElement;
  while (current) {
    const computedBg = window.getComputedStyle(current).backgroundColor;
    // "transparent" or "rgba(0, 0, 0, 0)" are both considered transparent
    if (computedBg && computedBg !== "transparent" && computedBg !== "rgba(0, 0, 0, 0)") {
      bg = computedBg;
      break;
    }
    current = current.parentElement;
  }
  if (!bg) {
    logger.info("No non-transparent background found in ancestors");
    return null;
  }
  change = {
    property: "color",
    before: element.style.color,
    after: bg
  };
  break;
}

```typescript
if (action === "commit-text") {
  if (!selectedElement || !panel) {
    return;
  }
  const newText = panel.getTextContent();
  const oldText = selectedElement.textContent ?? "";
  if (newText === oldText) {
    return;
  }
  selectedElement.textContent = newText;
  const patch: ContentPatch = {
    id: `${Date.now()}-${state.patches.length + 1}`,
    kind: "content",
    targetElement: selectedElement,
    targetDescriptor: describeElement(selectedElement),
    before: oldText,
    after: newText,
    createdAt: Date.now()
  };
  recordContentPatch(state, patch);
  history.undoStack.push(patch);
  history.redoStack.length = 0;
  logger.info("Text content changed", { target: patch.targetDescriptor });
  refreshHistoryButtons();
  return;
}
```
