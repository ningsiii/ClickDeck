✦ 适用场景与人群
PPT 演示 / 汇报 产品经理 (PM) UI 设计 / 排版 UX / 体验研究
Ask Gemini in Chrome
Google 官方说明

"Chat with Gemini while you browse to ask questions, summarize, and get insights about the page you're viewing."

相比于专注重写源码的 Coding AI（工程队），能看到真实网页的 Ask Gemini 是顶级的设计评审官。

视角不同，给出的价值截然不同。一键复制 ClickDeck 预设的 Prompt，让 AI 直接为你提供专业的页面级诊断。

01. 看逻辑

Prompt

看页面顺序、因果关系和叙事节奏。

💡 核心价值：提取网页真实的图文流，让 AI 评估叙事逻辑是否顺畅。不仅能找出转化漏斗里的逻辑断层，还能把冗长的网页快速梳理成干练的大纲。

02. 看重点

Prompt

看视觉主次，提醒哪些核心信息被埋没。

💡 核心价值：将页面当前的 CSS 表现带给模型，让它像设计总监一样指出视觉焦点的偏差，并给出对比度、字号或留白优化的具体修改建议。

03. 看交互

Prompt

看按钮、切换和滚动路径是不是顺手。

💡 核心价值：结合页面的真实 DOM 结构，评估用户的点击与操作体验是否符合直觉，提前发现容易造成流失的反人类摩擦点。

Export & Handoff

评审结束，然后呢？拿走结果，或交还给工程队

01

固化资产 (Export)

无缝导出静态 HTML、高清长图或 PDF，把网页当前的完美状态封存为确定的产物。

share & archive

适合页面已调顺，准备用于离线归档、对外分享或向客户做静态定稿交付。

02

登台演示 (Present)

一键进入沉浸式全屏放映模式，隐藏所有杂乱控件，让原生网页秒变高保真 PPT 舞台。

live pitch

适合直接带着修改后的原型上台提案、做工作汇报，或者录制产品宣发视频。

03

交还工程队 (Handoff)

不用手敲枯燥的重构需求。一键将视觉修改意图打包成极高精度的 Prompt 喂给大模型。
